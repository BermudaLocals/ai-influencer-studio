## 🎯 Quick Deploy (PowerShell):

### Step 1: Extract this ZIP
### Step 2: Open PowerShell in the extracted folder
### Step 3: Run these commands:

```powershell
# Initialize git repository
git init
git add .
git commit -m "Initial commit - AI Influencer Studio"

# Create GitHub repository (replace YOUR_USERNAME)
git remote add origin https://github.com/YOUR_USERNAME/ai-influencer-studio.git
git branch -M main
git push -u origin main
```

### Step 4: Connect to Railway

1. Go to https://railway.app/dashboard
2. Click "New Project" → "Deploy from GitHub repo"
3. Select your repository
4. Add PostgreSQL and Redis databases
5. Configure environment variables (see .env.example)
6. Deploy!
